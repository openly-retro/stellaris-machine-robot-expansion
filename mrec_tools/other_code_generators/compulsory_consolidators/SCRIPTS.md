# How to run the script generators

From `mre_dev_version/mrec_tools/other_code_generators/compulsory_consolidators/src/oxr_consolidators`, run:

	python main.py --feature-folder /home/user/modding/mre_dev_version/mrec_tools/features

Then, from the base folder (mre_dev_version), run:

	python mrec_tools/features/sync.py --feature compulsory_consolidators
